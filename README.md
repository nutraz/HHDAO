## ğŸŒ Overview

HeliosHash DAO (HHDAO) enables Indian citizens to co-own solar-powered Bitcoin mining parks. Our flagship pilot project in Baghpat District, Uttar Pradesh delivers:

- **Monthly INR profits** to 750 community members
- **Free electricity** to rural schools and hospitals  
- **Revenue share** to landowners (7% of net profit, no rent model)
- **100% solar powered** with zero electricity bills

## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
```bash
# Clone repository
git clone https://github.com/nutraz/HHDAO.git
cd HHDAO

# Install dependencies
npm install

# Set environment variables
cp .env.example .env.local
# Edit .env.local to add your Gemini API key (if needed)

# Start development server
npm run dev
```

### Build for Production
```bash
# Build project
npm run build

# Preview production build
npm run preview

# The built files will be in the 'dist' folder
```
